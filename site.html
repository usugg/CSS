🧪 Chrome DevTools Sensor Simulation

  1. Öffne Chrome DevTools (F12)
  2. Gehe zu "More Tools" → "Sensors"
  3. Wähle ein vordefiniertes Gerät oder "Custom orientation" für manuelle Tests
  4. Aktiviere den Data Collection Toggle unten
  5. Die simulierten Sensordaten werden in Echtzeit angezeigt

Hinweis: Diese Seite funktioniert am besten mit HTTPS oder localhost.
